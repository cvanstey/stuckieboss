Below are a few ways to optimize Elastic Cloud Compute (EC2) Costs
Ask me about AWS' Database Migration Programs and Services!

When the bill arrives for AWS EC2 instances, the Elastic Compute Cloud line item in your bill includes resources such as Elastic IP addresses and Amazon Elastic Block Store volumes (Amazon EBS).

Remember the following items:

  • Stopped instances don't incur charges, but Elastic IP addresses or EBS volumes attached to those instances do.
  • Use Spot Instances to reduce your computing costs when you have flexibility in when your applications can run.
  • Migrate workloads to smaller size R5b instances or consolidate workloads on fewer R5b instances to lower costs.
  • Consider Compute Savings Plans or EC2 Instance Savings Plans, two types of AWS Saving Plans.
